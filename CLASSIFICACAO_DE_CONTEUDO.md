# CLASSIFICAÇÃO DE CONTEÚDO

Todo arquivo sensível deve começar com classificação.

Modelo:
CLASSIFICAÇÃO: [PÚBLICO / INTERNO / SIGILOSO / SECRETO / COFRE / MILITAR-SAGRADO SENSÍVEL]
AUTORIZAÇÃO: [pública / equipe / fundador / cofre / revisão humana]
PUBLICAÇÃO: [pode publicar / não publicar / publicar somente versão sanitizada]
ACESSO: mínimo necessário
REVISÃO: humana obrigatória quando cabível
